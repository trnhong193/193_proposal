Scope of Work

viAct Responsibilities

  • Software: AI video analytics license, maintenance, and support
  • Camera integration: Integration with existing Verkada cameras via RTSP
  • Cloud infrastructure: Cloud-based deployment and management
  • System configuration: AI module configuration and calibration
  • Training: User training and documentation

Client Responsibilities

  • Hardware: Existing camera infrastructure (Verkada CD41 and CD41-E cameras)
  • Network: Stable internet connection with minimum 12 Mbps per camera, port 554 access
  • Power: Stable power source for cameras and network equipment
  • Speaker integration: Integration of existing on-site speakers (3 speakers per workshop) with the AI system
  • Site access: Provide access for site survey and system deployment
